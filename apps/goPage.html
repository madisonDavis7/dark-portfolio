A Game of Go Page

Java
Terminal Output
Game Research

Thi was the second project I completed in my intermediate Java class. In addition to still learning Java, I also had to learn the rules of Go, and there were many. it's a fairly complicated game and took a lot of planning and revising to get the logic down correctly. This was a terminal based app so I needed to figure out a way to clearly display the board, whos turn it was, pieces captured, and available spots on the board.

To make this game I used a lot of lits and strings. These came in very handy for keeping track of the player, moves, pieces on each side, and the board itself. Because there are so many rules I also used many conditional statements to ensure players were only making legal moves.
